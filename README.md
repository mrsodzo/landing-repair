# Ремонт квартир под ключ — Лендинг

Одностраничный лендинг для компании «РемСтрой» — профессиональный ремонт квартир под ключ.

## Цель

Конвертировать посетителей в заявки на бесплатный замер через форму, показать примеры работ и вовлечь калькулятором стоимости.

## Структура

1. Шапка с навигацией и телефоном
2. Hero-секция с заголовком и CTA
3. Интерактивный калькулятор стоимости
4. Блок услуг (демонтаж, черновые работы, чистовая отделка)
5. Галерея «до/после» с ползунком сравнения
6. Этапы работы (timeline)
7. Форма заявки на замер
8. Отзывы клиентов
9. FAQ-аккордеон
10. Футер с контактами

## Технологии

- HTML5
- CSS3 (flexbox, grid, clamp, media queries)
- Vanilla JavaScript (ES6+)
- Адаптивная вёрстка (mobile-first)
- Шрифты: system-ui

## Особенности

- Интерактивный калькулятор с пересчётом стоимости в реальном времени
- Слайдер сравнения «до/после» на чистом JS/CSS
- FAQ-аккордеон
- Мобильное меню
- Плавный скролл по якорям
- Оптимизированные изображения из Unsplash
